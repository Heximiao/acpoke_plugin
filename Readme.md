# NapCat 戳一戳插件 v0.4.1

本插件旨在为麦麦平台添加主动戳别人的功能。

---

## 功能说明

- 当前插件已经支持在所有群内发送戳一戳消息。

---

## 插件配置

1. 在 NapCat 中新建一个 HTTP 服务器。
2. 关闭 CORS 和 WebSocket 功能。
3. 设置地址（host）为 `127.0.0.1`，端口为 `4999`。

> ⚠️ 虽然 Adapter 在写该版本时已开放了 poke 的隧道，但本插件尚未支持该功能。

---

## 使用建议

- 强烈建议在更新插件前备份当前插件文件，以免意外丢失。
